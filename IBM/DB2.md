# DB2

IBM relational database engine for transactional, analytical, and enterprise workloads.

https://www.ibm.com/products/db2

## Position

- Enterprise SQL database.
- Compare with [Oracle Database](../Oracle/Oracle%20Database.md), [Microsoft SQL Server](../MS/SQLServ/Microsoft%20SQL%20Server.md), [PostgreSQL](../PostgreSQL/PostgreSQL.md), and [Teradata](../Teradata/Teradata.md).

## Notes

- Long-running enterprise database family across mainframe, Linux/Unix/Windows, and cloud offerings.
- Supports OLTP, warehousing, high availability, security, and mixed enterprise workloads.
- Important frontliner in IBM-heavy estates.
