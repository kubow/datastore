# ScyllaDB

High-performance distributed NoSQL wide-column database compatible with Apache Cassandra APIs.

https://www.scylladb.com/

## Position

- Cassandra-compatible operational NoSQL database.
- Compare with [Apache Cassandra](../Apache/Apache%20Cassandra.md), [Apache HBase](../Apache/Apache%20HBase.md), [DynamoDB](../Amazon/AWS%20DynamoDB.md), and [Couchbase](../Couchbase/Couchbase.md).

## Notes

- Built for low-latency, high-throughput workloads.
- Uses a shard-per-core architecture.
- Common when teams want Cassandra-style data modeling with stronger performance predictability.
