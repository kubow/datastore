# Weaviate

Vector database for semantic search and AI-native applications.

https://weaviate.io/

## Position

- Vector database and search platform.
- Compare with [Milvus](../Milvus/Milvus.md), [Qdrant](../Qdrant/Qdrant.md), Elasticsearch/OpenSearch vector search, and PostgreSQL with pgvector.

## Notes

- Stores objects plus vector embeddings.
- Supports hybrid search combining keyword and vector retrieval.
- Common in RAG, semantic search, recommendation, and classification systems.
