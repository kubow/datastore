# StarRocks

MPP analytical database for high-performance SQL analytics, real-time reporting, and lakehouse query acceleration.

https://www.starrocks.io/

## Position

- Frontline OLAP database in the same broad space as [ClickHouse](../ClickHouse/ClickHouse.md), [Apache Doris](../Apache/Apache%20Doris.md), [Druid](../Apache/Apache%20Druid.md), [Pinot](../Apache/Apache%20Pinot.md), and [DuckDB](../Duck/DuckDB.md).
- Often used as a serving layer for dashboards and user-facing analytics.

## Notes

- Columnar MPP execution.
- Supports fast aggregation, joins, materialized views, and lakehouse-style external data access.
- Useful when dashboard concurrency and low query latency matter.
