# Prometheus

Open-source monitoring and time-series database system.

https://prometheus.io/

## Position

- Time-series storage and metrics query engine.
- Compare with [InfluxDB](../InfluxDB/InfluxDB.md), [TDengine](../TDEngine/TDEngine.md), [TimescaleDB](../TimescaleDB/TimescaleDB.md), and [QuestDB](../QuestDB/QuestDB.md).

## Notes

- Pull-based metrics collection model.
- PromQL query language.
- Common default for Kubernetes and cloud-native metrics monitoring.
