# Airbyte

Open-source data movement platform for extracting and loading data from applications, APIs, databases, and files.

https://airbyte.com/

## Position

- ELT/data integration tool.
- Compare with [Fivetran](../Fivetran/Fivetran.md), [Meltano](../meltano.md), [dlt](../dlt.md), [Estuary](../Estuary.md), and [Qlik Data Integration](../Qlik/Qlik%20Data%20Integration.md).

## Notes

- Connector-driven ingestion.
- Used to move source data into warehouses, lakes, and lakehouses.
- Open-source core plus managed cloud offering.
