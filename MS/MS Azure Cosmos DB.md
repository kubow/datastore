# Azure Cosmos DB

Globally distributed multi-model NoSQL database service on Microsoft Azure.

https://azure.microsoft.com/en-us/products/cosmos-db/

## Position

- Cloud-native operational NoSQL database.
- Compare with [DynamoDB](../Amazon/AWS%20DynamoDB.md), [MongoDB](../MongoDB/MongoDB.md), [Cassandra](../Apache/Apache%20Cassandra.md), [Firestore](../Google/Google%20Firestore.md), and [Cloud Spanner](../Google/Google%20Cloud%20Spanner.md).

## Notes

- Supports multiple APIs, including NoSQL/document, MongoDB-compatible, Cassandra-compatible, Gremlin, and Table-style access.
- Designed for global distribution, low latency, and elastic throughput.
- Common in Azure application architectures that need managed distributed storage.
