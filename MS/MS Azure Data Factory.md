# Azure Data Factory

Managed data integration and orchestration service on Microsoft Azure.

https://azure.microsoft.com/en-us/products/data-factory/

## Position

- Cloud ETL/ELT, pipeline orchestration, and data movement service.
- Compare with [AWS Glue](../Amazon/AWS%20Glue.md), [Google Dataflow](../Google/Google%20Dataflow.md), [Airbyte](../Airbyte/Airbyte.md), [Fivetran](../Fivetran/Fivetran.md), and [Apache NiFi](../Apache/Apache%20NiFi.md).

## Notes

- Uses pipelines, activities, linked services, datasets, and integration runtimes.
- Common in Azure estates for moving data among SQL Server, Synapse, ADLS, Fabric, SaaS tools, and other systems.
- Often paired with Azure Synapse Analytics, Azure SQL, ADLS, and Power BI.
