# Exasol

Analytical in-memory MPP database for data warehouse and BI workloads.

https://www.exasol.com/

## Position

- Data warehouse / OLAP database.
- Compare with [Snowflake](../Snowflake/Snowflake.md), [Teradata](../Teradata/Teradata.md), [Vertica](../Vertica/Vertica.md), [ClickHouse](../ClickHouse/ClickHouse.md), and [SingleStore](../SingleStore/SingleStore.md).

## Notes

- SQL analytical engine with columnar storage and MPP execution.
- Often positioned for high-performance BI, reporting, and analytical workloads.
- Supports deployment in cloud and self-managed environments.
