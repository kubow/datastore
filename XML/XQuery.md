[W3C XML Query (XQuery)](https://www.w3.org/XML/Query/)

XQuery is a query and construction language for XML values and XML document collections. It is closest to "SQL for XML", but its data model is ordered trees and sequences rather than tables.

## Where It Fits

- Query native XML databases such as BaseX, eXist-db and Sedna.
- Query XML columns in XML-capable relational databases.
- Extract, join and reshape XML collections.
- Generate XML, HTML, text or JSON-like output from XML sources.

## Core Concepts

- FLWOR expressions: `for`, `let`, `where`, `order by`, `return`.
- XPath expressions for node selection.
- Construction of new elements and attributes.
- Sequence processing rather than row processing.
- Optional schema-aware typing when XSD information is available.
