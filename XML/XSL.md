
## XSLT

XSLT transforms XML trees into XML, HTML, text or other structured output. It is declarative and template-based: rules match input nodes and produce output nodes.

## XSL Family

- **XSLT**: transformation language.
- **XPath**: expression and node selection language used by XSLT.
- **XSL-FO**: formatting objects for paged output such as PDF; less common in modern stacks.

## Typical Use

- Convert XML to HTML or another XML vocabulary.
- Publish XML documents into reports or static pages.
- Normalize vendor XML into a canonical model.
- Extract structured text from XML documents.

## Engines

- Saxon is a major XSLT/XQuery processor.
- Xalan and libxslt are common older XSLT processors.
- Many databases and programming runtimes embed XSLT support.
