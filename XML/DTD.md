Document Type Definition (DTD) defines the allowed structure and elements of an XML document.

## What DTD Covers

- Element declarations and allowed child order.
- Attribute declarations and default values.
- Entity declarations.
- ID and IDREF references.
- Document type declaration through `<!DOCTYPE ...>`.

## Limits

- Weak data typing compared to [XSD](XSD.md).
- Poor namespace support.
- Less suitable for modern service contracts and mixed vocabularies.
- External DTD loading can create security risk when parsers allow external entities.
