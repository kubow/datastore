# XML Schema Definition

XSD (XML Schema Definition) is the W3C schema language for validating XML document structure and data types.

## What XSD Adds

- Namespace-aware validation.
- Strong scalar data types: string, decimal, integer, date, dateTime, boolean and derived types.
- Cardinality through `minOccurs` and `maxOccurs`.
- Reusable complex types and simple types.
- Constraints such as enumeration, pattern, length, range, `unique`, `key` and `keyref`.
- Better support for data exchange contracts than DTD.
