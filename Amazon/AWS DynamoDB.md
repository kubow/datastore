# Amazon DynamoDB

Managed serverless NoSQL key-value and document database on AWS.

https://aws.amazon.com/dynamodb/

## Position

- Cloud-native operational database.
- Compare with [Cassandra](../Apache/Apache%20Cassandra.md), [MongoDB](../MongoDB/MongoDB.md), [Cosmos DB](../MS/MS%20Azure%20Cosmos%20DB.md), [Firestore](../Google/Google%20Firestore.md), and [Redis](../REDIS/Redis.md).

## Notes

- Designed for high-scale, low-latency operational workloads.
- Uses partition keys and optional sort keys as the core data model.
- Common in serverless and event-driven AWS architectures.
