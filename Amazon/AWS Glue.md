# AWS Glue

Managed serverless data integration, ETL, and metadata catalog service on AWS.

https://aws.amazon.com/glue/

## Position

- Cloud data integration and catalog service.
- Compare with [Azure Data Factory](../MS/MS%20Azure%20Data%20Factory.md), [Google Dataflow](../Google/Google%20Dataflow.md), [Airbyte](../Airbyte/Airbyte.md), [Fivetran](../Fivetran/Fivetran.md), and [Apache NiFi](../Apache/Apache%20NiFi.md).

## Notes

- Includes Glue Data Catalog, crawlers, ETL jobs, and integration with Spark-based processing.
- Frequently used with S3 data lakes, Athena, Redshift, EMR, and Lake Formation.
- Important AWS frontliner because it often becomes the metadata/catalog layer for lakehouse query tools.
