# TimescaleDB

Time-series database built as a PostgreSQL extension.

https://www.timescale.com/

## Position

- Time-series database with PostgreSQL compatibility.
- Compare with [InfluxDB](../InfluxDB/InfluxDB.md), [Prometheus](../Prometheus/Prometheus.md), [QuestDB](../QuestDB/QuestDB.md), and [TDengine](../TDEngine/TDEngine.md).

## Notes

- Uses hypertables to partition time-series data.
- Keeps SQL and much of the PostgreSQL ecosystem available.
- Common for metrics, IoT, financial ticks, event data, and operational analytics.
