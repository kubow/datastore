# Apache HBase

Distributed wide-column database built on top of the Hadoop ecosystem.

https://hbase.apache.org/

## Position

- NoSQL/wide-column storage.
- Compare with [Cassandra](./Apache%20Cassandra.md), [ScyllaDB](../ScyllaDB/ScyllaDB.md), [DynamoDB](../Amazon/AWS%20DynamoDB.md), and [Apache Kudu](./Apache%20Kudu.md).

## Notes

- Inspired by Google's Bigtable model.
- Runs on HDFS or compatible storage in Hadoop-style deployments.
- Useful for sparse, large-scale tables with fast key/range access.
