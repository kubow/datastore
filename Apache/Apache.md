# Apache Software Foundation (ASF)

## Data And Query Engines

- [Apache Beam](Apache%20Beam.md) - unified batch and streaming processing model
- [Apache Calcite](Apache%20Calcite.md) - query planning and optimization framework
- [Apache Doris](Apache%20Doris.md) - real-time analytical database
- [Apache Drill](Apache%20Drill.md) - schema-free SQL query engine
- [Apache Druid](Apache%20Druid.md) - real-time columnar analytics database
- [Apache Flink](Apache%20Flink.md) - stateful stream processing engine
- [Apache Hadoop](Apache%20Hadoop.md) - distributed storage and processing ecosystem
- [Apache Hive](Apache%20Hive.md) - SQL data warehouse layer for Hadoop-style data lakes
- [Apache Spark](Apache%20Spark.md) - engine for large-scale data engineering, analytics, and ML
- [Apache Zeppelin](Apache%20Zeppelin.md) - notebook for interactive analytics

## Storage And Table Formats

- [Apache Arrow](Apache%20Arrow.md) - in-memory analytics columnar format
- [Apache Avro](Apache%20Avro.md) - data serialization system
- [Apache Cassandra](Apache%20Cassandra.md) - distributed wide-column NoSQL database
- [Apache CouchDB](Apache%20CouchDB.md) - JSON document database
- [Apache Derby](Apache%20Derby.md) - embedded relational database
- [Apache HBase](Apache%20HBase.md) - distributed wide-column database on Hadoop
- [Apache Hudi](Apache%20Hudi.md) - lakehouse table format for incremental data
- [Apache Iceberg](Apache%20Iceberg.md) - open table format for analytic datasets
- [Apache Kudu](Apache%20Kudu.md) - mutable columnar analytical storage
- [Apache Paimon](Apache%20Paimon.md) - lakehouse table format
- [Apache Parquet](Apache%20Parquet.md) - column-oriented data file format

## Streaming And Messaging

- [Apache Kafka](Apache%20Kafka.md) - event streaming platform
- [Apache Pulsar](Apache%20Pulsar.md) - distributed messaging and streaming
- [Apache Zookeeper](Apache%20Zookeeper.md) - coordination service

## Search, Analytics, And Applications

- [Apache Airflow](Apache%20Airflow.md) - workflows and scheduling
- [Apache NiFi](Apache%20NiFi.md) - flow-based data movement
- [Apache Pinot](Apache%20Pinot.md) - real-time OLAP datastore
- [Apache Presto](Apache%20Presto.md) - distributed SQL query engine
- [Apache Solr](Apache%20Solr.md) - search platform
- [Apache Submarine](Apache%20Submarine.md) - machine learning platform
- [Apache Superset](Apache%20Superset.md) - data exploration and visualization
- [Apache TinkerPop](Apache%20TinkerPop.md) - graph computing framework
