# Apache Solr

Open-source enterprise search platform built on Apache Lucene.

https://solr.apache.org/

## Position

- Search engine and indexing platform.
- Compare with [Elasticsearch](../Elasticsearch/Elasticsearch.md) and [OpenSearch](../OpenSearch/OpenSearch.md).

## Notes

- Provides full-text search, faceting, filtering, highlighting, and distributed search.
- Strong historical footprint in enterprise search.
- Often used when search relevance, indexing control, and Lucene-level capabilities matter.
