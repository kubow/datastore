# Apache Pinot

Distributed real-time OLAP datastore for low-latency analytics on high-throughput event data.

https://pinot.apache.org/

## Position

- Real-time analytics database.
- Compare with [Apache Druid](./Apache%20Druid.md), [ClickHouse](../ClickHouse/ClickHouse.md), [Apache Doris](./Apache%20Doris.md), and [StarRocks](../StarRocks/StarRocks.md).

## Notes

- Designed for user-facing analytics with high concurrency and low latency.
- Commonly ingests from Kafka and batch storage.
- Uses segments, indexes, and distributed query execution.
