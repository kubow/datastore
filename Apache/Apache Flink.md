# Apache Flink

Distributed stream processing engine for stateful event-time workloads, with batch support through the same execution model.

https://flink.apache.org/

## Position

- Frontline engine for low-latency stream processing.
- Compare with [Apache Spark](./Apache%20Spark.md), especially Spark Structured Streaming.
- Commonly paired with [Apache Kafka](./Apache%20Kafka.md), [Apache Pulsar](./Apache%20Pulsar.md), Iceberg, Hudi, Delta Lake, and data lakehouse pipelines.

## Notes

- Stateful operators with checkpointing and savepoints.
- Event-time processing with watermarks.
- Used for streaming ETL, fraud detection, feature pipelines, CDC processing, and operational analytics.
