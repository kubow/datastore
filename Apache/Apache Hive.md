# Apache Hive

SQL data warehouse layer for querying large datasets in Hadoop-style data lakes.

https://hive.apache.org/

## Position

- Data warehouse/query layer in the Hadoop ecosystem.
- Compare with [Trino](../Trino/Trino.md), [Presto](./Apache%20Presto.md), [Spark SQL](./Apache%20Spark.md), [BigQuery](../Google/Google%20BigQuery.md), and [Dremio](../Dremio/Dremio.md).

## Notes

- Provides SQL access over files and tables managed through a metastore.
- Historically central to Hadoop data warehouses.
- Still relevant through Hive Metastore compatibility used by many lakehouse and query engines.
