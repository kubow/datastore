# Apache Doris

Distributed real-time analytical database for high-concurrency OLAP and serving analytics.

https://doris.apache.org/

## Position

- MPP analytical database.
- Compare with [ClickHouse](../ClickHouse/ClickHouse.md), [Apache Druid](./Apache%20Druid.md), [Apache Pinot](./Apache%20Pinot.md), [StarRocks](../StarRocks/StarRocks.md), and [Exasol](../Exasol/Exasol.md).

## Notes

- SQL-oriented analytics with columnar storage.
- Common use cases include dashboards, ad-hoc analytics, real-time reporting, and data serving.
- Supports data loading from batch and streaming sources.
