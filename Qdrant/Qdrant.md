# Qdrant

Vector database and similarity search engine.

https://qdrant.tech/

## Position

- Vector storage and search engine.
- Compare with [Milvus](../Milvus/Milvus.md), [Weaviate](../Weaviate/Weaviate.md), Elasticsearch/OpenSearch vector search, and PostgreSQL with pgvector.

## Notes

- Stores embeddings with payload metadata.
- Supports filtering plus vector similarity search.
- Common fit for RAG, recommendation, semantic search, and matching systems.
