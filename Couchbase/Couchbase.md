# Couchbase

Distributed NoSQL document database with key-value access, SQL-like querying, caching, search, and mobile sync options.

https://www.couchbase.com/

## Position

- Operational NoSQL database.
- Compare with [MongoDB](../MongoDB/MongoDB.md), [CouchDB](../Apache/Apache%20CouchDB.md), [DynamoDB](../Amazon/AWS%20DynamoDB.md), [Cosmos DB](../MS/MS%20Azure%20Cosmos%20DB.md), and [Redis](../REDIS/Redis.md).

## Notes

- JSON document storage with key-value access.
- SQL++ query language for document data.
- Often considered when low-latency operational access and flexible JSON modeling are both needed.
