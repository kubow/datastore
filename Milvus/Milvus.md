# Milvus

Open-source vector database for similarity search and retrieval workloads.

https://milvus.io/

## Position

- Vector storage and search engine.
- Compare with [Weaviate](../Weaviate/Weaviate.md), [Qdrant](../Qdrant/Qdrant.md), Elasticsearch vector search, OpenSearch vector search, and PostgreSQL with pgvector.

## Notes

- Stores embeddings and metadata for semantic search, RAG, recommendation, and anomaly detection.
- Supports approximate nearest neighbor indexing.
- Often deployed with object storage and a metadata/logging backend depending on scale and architecture.
