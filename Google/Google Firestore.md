# Cloud Firestore

Managed serverless document database from Google Cloud and Firebase.

https://firebase.google.com/products/firestore

## Position

- Cloud-native document database.
- Compare with [MongoDB](../MongoDB/MongoDB.md), [DynamoDB](../Amazon/AWS%20DynamoDB.md), [Cosmos DB](../MS/MS%20Azure%20Cosmos%20DB.md), and [Cloud Spanner](./Google%20Cloud%20Spanner.md).

## Notes

- Stores data as documents organized into collections.
- Common in web, mobile, Firebase, and serverless application backends.
- Provides real-time synchronization and managed scaling.
