# Cloud Spanner

Globally distributed relational database service from Google Cloud.

https://cloud.google.com/spanner

## Position

- Cloud-native SQL database for transactional workloads.
- Compare with [CockroachDB](../CockroachDB/CockroachDB.md), [Amazon Aurora](../Amazon/AWS%20Aurora.md), [Cosmos DB](../MS/MS%20Azure%20Cosmos%20DB.md), and [PostgreSQL](../PostgreSQL/PostgreSQL.md).

## Notes

- Strong consistency with horizontal scale.
- Supports relational schema, SQL, transactions, and managed global replication.
- Common fit when operational data needs regional or global distribution without manual sharding.
