# Google Dataflow

Managed stream and batch data processing service on Google Cloud.

https://cloud.google.com/dataflow

## Position

- Managed runner for Apache Beam pipelines.
- Compare with [Apache Beam](../Apache/Apache%20Beam.md), [Apache Flink](../Apache/Apache%20Flink.md), [Apache Spark](../Apache/Apache%20Spark.md), [AWS Glue](../Amazon/AWS%20Glue.md), and [Azure Data Factory](../MS/MS%20Azure%20Data%20Factory.md).

## Notes

- Used for streaming ETL, batch pipelines, event processing, and data movement into BigQuery or storage.
- Built around Beam concepts such as windows, triggers, and event time.
- Commonly used with Pub/Sub, BigQuery, Cloud Storage, and Dataflow templates.
